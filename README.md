# 🍰 Honey & Flour – Bakery Menu Web App

An elegant and interactive **Bakery Menu Web Application** built using **HTML, CSS, and JavaScript**.
This project simulates a modern bakery ordering experience with a beautiful UI and dynamic cart system.

---

## ✨ Features

* 🧁 Interactive menu with categories (Cakes, Pastries, Breads, Drinks)
* 🛒 Fully functional cart system
* ➕ Add / ➖ Remove / 🗑️ Delete items
* 💰 Real-time total price calculation
* 🔔 Toast notifications for actions
* 📱 Responsive design (mobile-friendly)
* 🎨 Aesthetic and modern UI design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS, DOM Manipulation)
